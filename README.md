# 美心西饼优惠券 - Apple Wallet Demo

这是一个美心西饼优惠券的网页演示，支持添加到 Apple Wallet。

## 在线访问

如果你想在手机上测试，可以：

1. 将这个项目上传到 GitHub
2. 启用 GitHub Pages
3. 通过 HTTPS 访问

## 本地测试

在 Mac 上：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`

## 文件说明

- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - JavaScript 功能
- `Pass/maxims-coupon.pkpass` - Apple Wallet Pass 文件
